Результаты тестирования в Geekbench 5 и сравнение общих данных

Результаты тестирования процессора в Geekbench 5 в операционной системе Windows 10 установленной на компьютер: производительность на одно ядро составила 876 баллов, на все 4 ядра - 2988 баллов.

Результаты тестирования процессора в операционных системах запущенных через VirtualBox:
Windows 10 - баллов производительности на одно ядро - 818, на все 4 ядра - 2676.
Ubuntu - баллов производительности на одно ядро - 838, на все 4 ядра - 2359.
macOS - баллов производительности на одно ядро - 810, на все 4 ядра - 2258.

Исходя из представленных результатов, лучшую производительность процессор выдавал под управлением ОС Windows 10. Ни одна из виртуальных машин не смогла достигнуть результата работы процессора такого же, как при взаимодействии с операционной системой напрямую. По опыту использования каждой из операционных систем, лучше всего себя показала Windows 10. Она быстрее всего запускала программы.

Быстрее всего установилась Linux Ubuntu, её инсталяция заняла всего 13 минут. Тогда как инсталяция Windows заняла 16 минут, после запуска требовалась дополнительная настройка, тогда как в Linux Ubuntu основная настройка производится в процессе установки. Установка macOS заняла 20 минут времени. Установка Linux Ubuntu является самой простой из представленных операционных систем, её без труда сможет поставить любой пользователь ПК. Кроме этого, она предоставляет возможность ознакомиться с операционной системой до её установки, с помощью live-cd образа.

После установки, самой заполненной операционной системой является Windows 10. Очень многое из предустановленного ПО не нужно конечному пользователю в домашней версии системы. Например, она вся ещё поставляется в комплекте с уже устаревшим интернет-браузером Internet Explorer и не устаревшим интернет-браузером Microsoft Edge. В комплекте же с Ubuntu идёт браузер Firefox, а macOS - Safari.

С точки зрения загрузки и лёгкости установки приложений, самой лёгкой в освоении и доступностью для пользователя является Windows 10, так как она обладает максимально широким спектром приложений установка которых является максимально простой. Сложнее с этим дела обстоят у macOS, так как спектр приложений более ограниченный, и тяжелее всего на Linux Ubuntu, так как часть программ могут быть лишены графического интерфейса, который имеется в этой же программе на Windows (Geekbench), кроме этого, куда сложнее установка, так как может потребовать использование командной строки.
